⚡ Functions (Funciones Globales)

Funciones utilitarias básicas disponibles en todos los scripts

exit()

Definición JavaScript
function exit(reason?: string): void;

Descripción: Termina la ejecución del script anticipadamente, útil para salir basado en ciertas condiciones.

Parámetros

ParámetroTipoDescripción
reason?stringRazón opcional del por qué falló el script, aparece en el área de consola
Ejemplo
if (player.isDead) { exit("Failed, I'm dead!"); }

log()

Definición JavaScript
function log(...args: any[]): void;

Descripción: Registra los argumentos en el área de consola debajo de la ventana de scripting.

Parámetros

ParámetroTipoDescripción
...argsany[]Lista de objetos que se registrarán en la consola
Ejemplos
log('Hi there!'); log(`My name is ${player.name}`); log(`My helmet is`, player.equippedItems.helmet);

sleep()

Definición JavaScript
function sleep(ms: number): void;

Descripción: Retrasa la ejecución del script durante un número específico de milisegundos.

⚠️ Nota: La precisión del retraso para duraciones largas no está garantizada.

Parámetros

ParámetroTipoDescripción
msnumberRetraso en milisegundos
Ejemplo
player.useSkill(Skills.Anatomy); sleep(10000); // dormir 10 segundos player.useSkill(Skills.Meditation);

🌐 Variables Globales

Objetos principales disponibles para interactuar con el juego

client : Client

El objeto principal para interactuar con el cliente ClassicUO y el mundo del cliente.

player : Player

Representa al jugador actual mientras está en el juego.

target : Target

Controla el sistema de targeting del juego.

journal : Journal

Proporciona acceso al sistema de journal del juego para activar partes de tu script.

ignoreList : IgnoreList

Administra una lista de entidades ignoradas dentro del cliente web ClassicUO.

popupMenu : PopupMenu

Maneja los menús contextuales que aparecen cuando haces single click en objetos.

prompt : Prompt

Para interactuar con prompts de texto vía chat box.

worldMap : WorldMap

Controla las funcionalidades relacionadas con el mapa del mundo.

Métodos de Búsqueda de Objetos

findObject()

Definición JavaScript
findObject( serial: number | SerialObject | GameObject | "world", hue?: null | number, sourceSerial?: null | number | SerialObject | GameObject | "world", amount?: null | number, range?: null | number ): any

Descripción: Intenta verificar si un cierto objeto puede encontrarse en el juego.

Ejemplo
const runebookSerial = 0x401c37fb; const runebook = client.findObject(runebookSerial); if (runebook) { player.use(runebook); } else { client.headMsg('Runebook missing!', player.serial); }

findType()

Definición JavaScript
findType( graphic: number, hue?: null | number, sourceSerial?: null | number | SerialObject | GameObject | "world", amount?: null | number, range?: null | number ): any

Descripción: Intenta encontrar un objeto en el mundo con los parámetros de búsqueda especificados, retornándolo si se encuentra.

ℹ️ Nota: El parámetro range cuando se combina con el parámetro de contenedor source especifica la profundidad de búsqueda.
Ejemplos
// Usar cualquier vendaje que se pueda encontrar const bandageType = 0xe21; const bandages = client.findType(bandageType); if (bandages) { player.use(bandages); target.waitTargetSelf(); } else { client.headMsg('Out of bandages', player.serial); }

📜 Namespace: Journal

Proporciona acceso al sistema de journal del juego para activar partes de tu script

clear()

Definición JavaScript
clear(): void

Descripción: Limpia/olvida el historial del journal. Útil si no quieres procesar entradas más antiguas del journal.

Ejemplo
player.say('Hello there'); journal.waitForText('Hello there'); // Éxito rápido journal.clear(); journal.waitForText('Hello there'); // Timeout, string limpiado

containsText()

Definición JavaScript
containsText(text: string, author?: string): boolean

Descripción: Verifica la existencia de texto en el journal.

Retorna: true/false si el texto existe en el journal.

waitForText()

Definición JavaScript
waitForText(text: string, author?: string, timeout?: number): boolean

Descripción: Espera a que aparezca el texto en el journal.

Nota: Usa clear() para limpiar el texto del journal.

Retorna: true/false si el texto fue encontrado.

Ejemplo
const healthBefore = player.hits; journal.clear(); player.useType(0xe21); target.waitTargetSelf(); if (journal.waitForText('You healed', 'System', 8000)) { client.headMsg(`Bandaged +${player.hits - healthBefore}`, player, 66); }

waitForTextAny()

Definición JavaScript
waitForTextAny(text: string[], author?: string, timeout?: number): null | string

Descripción: Espera a que CUALQUIER string en un array de strings sea encontrado en el Journal, o a que se alcance el timeout. Retorna el primer string encontrado o null si se alcanzó el timeout.

Uso: Útil si estás esperando cualquiera de una lista de strings que aparezca. Si quieres que todos los strings aparezcan deberías usar waitForTextEvery.

Ejemplo
const waitMessage = 'You must wait'; const failMessage = 'You cannot focus'; const successMessage = 'You enter a meditative trance.'; journal.clear(); const response = journal.waitForTextAny([waitMessage, failMessage, successMessage]); switch (response) { case waitMessage: // sleep break; case failMessage: // retry break; case successMessage: // loop break; }

waitForTextEvery()

Definición JavaScript
waitForTextEvery(text: string[], author?: string, timeout?: number): string[]

Descripción: Espera a que TODOS los strings en un array de strings sean encontrados en el Journal, o a que se alcance el timeout. Retorna todos los strings que fueron encontrados, puede ser menos que la entrada si se alcanzó el timeout.

Uso: Útil si estás esperando que varios strings aparezcan todos. Si quieres esperar cualquier string del array de entrada usa waitForTextAny.

Retorna: Todos los strings que fueron encontrados dentro del timeout, o un array vacío si ninguno fue encontrado.

Ejemplo
const waitMessage = 'You must wait'; const failMessage = 'You cannot focus'; const successMessage = 'You enter a meditative trance.'; journal.clear(); const response = journal.waitForTextEvery([waitMessage, failMessage, successMessage]); if (response.includes(successMessage)) { client.headMsg(`Meditating...`, player, 66); }

function autoHeal() { const healthPercent = (player.hits / player.maxHits) * 100; if (healthPercent < 70) { const bandages = client.findType(0xe21); // Bandage type if (bandages) { const healthBefore = player.hits; journal.clear(); player.use(bandages); target.waitTargetSelf(); if (journal.waitForText('You healed', 'System', 8000)) { client.headMsg(`Healed +${player.hits - healthBefore}`, player, 66); } else { log('Healing failed or interrupted'); } } else { client.headMsg('Out of bandages!', player, 33); } } }

Recolección Automatizada de Oro

Script Completo
function collectGold() { const goldType = 0xeed; ignoreList.clear(); let goldCount = 0; let totalAmount = 0; let gold: Item; while ((gold = client.findType(goldType, undefined, 'world') as Item)) { if (gold.amount > 0) { totalAmount += gold.amount; player.moveItem(gold, player.backpack); goldCount++; ignoreList.add(gold); sleep(500); // Evitar spam } } client.headMsg(`Collected ${goldCount} piles (${totalAmount} gold)`, player, 66); ignoreList.clear(); }

Sistema de Trading Automatizado

Script Avanzado
function autoSellWeapons() { player.say('vendor sell'); const sellData = Gump.waitForVendorGumpData(5000); if (sellData && sellData.type === 'sell') { // Vender todas las armas excepto la equipada const weaponsToSell = sellData.items.filter(item => isWeapon(item.graphic) && item.serial !== player.equippedItems.oneHanded?.serial && item.serial !== player.equippedItems.twoHanded?.serial ); if (weaponsToSell.length > 0) { client.sendSellRequest(sellData.vendor, weaponsToSell); const totalValue = weaponsToSell.reduce((sum, item) => sum + item.price, 0); client.headMsg(`Sold ${weaponsToSell.length} weapons for ${totalValue} gold`, player, 66); } else { client.headMsg('No weapons to sell', player, 33); } } else { exit('Vendor sell gump not found'); } } function isWeapon(graphic: number): boolean { const weaponTypes = [0x0f52, 0x0f49, 0x13b2, 0x0f5c, 0x0f61, 0x1441]; return weaponTypes.includes(graphic); }

Script de Entrenamiento de Skills

Sistema de Skill Training
function trainAnatomy() { const skill = player.getSkill(Skills.Anatomy); if (!skill) { exit('Cannot get anatomy skill info'); } const currentSkill = skill.value / 10; // Convertir de 746 a 74.6 const targetSkill = 100.0; if (currentSkill >= targetSkill) { client.headMsg('Anatomy training complete!', player, 66); return; } client.headMsg(`Training Anatomy: ${currentSkill.toFixed(1)}%`, player, 66); // Encontrar target para anatomía const nearestMobile = client.selectEntity( SearchEntityOptions.Any, SearchEntityRangeOptions.Nearest, SearchEntityTypeOptions.Any, false ); if (nearestMobile) { journal.clear(); player.useSkill(Skills.Anatomy); target.waitTargetEntity(nearestMobile, 5000); // Esperar resultado const result = journal.waitForTextAny([ 'You learn', 'You fail', 'You must wait' ], undefined, 5000); if (result?.includes('You must wait')) { sleep(10000); // Esperar cooldown } else if (result?.includes('You learn')) { const newSkill = player.getSkill(Skills.Anatomy).value / 10; client.headMsg(`Gained! Now: ${newSkill.toFixed(1)}%`, player, 66); } sleep(1000); // Pausa entre usos } else { client.headMsg('No targets found for training', player, 33); sleep(5000); } }

// Limpiar journal antes de acciones críticas journal.clear(); player.useType(0xe21); // Vendajes target.waitTargetSelf(); // Usar múltiples mensajes posibles const healMessages = ['You healed', 'You heal', 'You apply']; const failMessages = ['You fail', 'cannot be healed', 'You must wait']; const result = journal.waitForTextAny([...healMessages, ...failMessages], 'System', 8000); if (healMessages.some(msg => result?.includes(msg))) { client.headMsg('Healing successful', player, 66); } else if (failMessages.some(msg => result?.includes(msg))) { client.headMsg('Healing failed or interrupted', player, 33); } else { log('No healing message detected, possible timeout'); }

Template Recomendado
/** * Script Template Profesional para ClassicUO * Descripción: [Descripción del script] * Autor: [Tu nombre] * Versión: 1.0 */ // Configuración del script const CONFIG = { DEBUG: false, DELAY_MS: 500, TIMEOUT_MS: 5000, MAX_ITERATIONS: 100 }; // Función principal function main() { try { log('Starting script...'); // Verificaciones iniciales if (!performInitialChecks()) { exit('Initial checks failed'); } // Loop principal let iterations = 0; while (iterations < CONFIG.MAX_ITERATIONS) { if (!performMainLogic()) { break; // Condición de salida } iterations++; sleep(CONFIG.DELAY_MS); } log(`Script completed after ${iterations} iterations`); } catch (error) { log(`Fatal error: ${error}`); exit(`Error: ${error}`); } finally { cleanup(); } } function performInitialChecks(): boolean { if (!player || player.hits <= 0) { log('Player not valid or dead'); return false; } if (!player.backpack) { log('Cannot access backpack'); return false; } return true; } function performMainLogic(): boolean { // Lógica principal del script aquí if (CONFIG.DEBUG) { log('Executing main logic...'); } // Retornar false para terminar el loop return true; } function cleanup() { // Limpiar recursos ignoreList.clear(); Gump.closeAll(); log('Cleanup completed'); } // Iniciar el script main();
